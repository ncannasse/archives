Conclusion

If we jump fast to conclusions, we can say that :

  • MotionScript is 25 times faster than ActionScript
  • MotionScript is 2.3 times faster than Python
  • Java is faster than MotionScript (Just-In-Time helps here)

HOWEVER, theses benchmarks are micro-benchmarks, and should be taken with care. Jumping to false conclusions is easy, but what we can say without a doubt is that :

  • MotionScript and Python are two efficient languages
  • ActionScript is very very slow
  • Java JIT works well on the machine we did the benchmarks
