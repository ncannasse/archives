MTS Math Library :

Mathematics functions :
sqrt(x) return square root of x.
cos(x) return cosinus of x.
sin(x) return sinus of x.
tan(x) return tangent of x.
log(x) return logarithm of x.
exp(x) return exponantial of x.
pow(x,y) return x at power y.
atan(x) return arc-tangent of x.
atan2(y,x) return arc-tangent of y/x.
acos(x) return arc-cosinus of x.
asin(x) return arc-sinus of x.
ceil(x) return integer value of x rounded to superior.
floor(x) return integer value of x rounded to inferior.
round(x) return integer value of x rounded to nearest.
abs(x) return absolute value of x.
pi PI value.
to_angle(x) normalize x value between 0 and 2PI excluded.
