Welcome

The Honsen Project is an Internet browser plugin that enables you to easily develop rich interactive 3D applications and games for the web. It features :

  • DirectX based 3D engine.
  • Motion-Script scripting language (benchmarks).
  • Free development tools and player !

Developers : Get Started

Want to give Honsen a try ? Check out the Samples section to see what can kind of things can be done using Honsen. Then you can download the Free SDK including tutorials and complete documentation to start building your own Honsen content ! Join the Honsen Community to talk about Honsen development, evolution, and share knowledge with other Honsen developers.
