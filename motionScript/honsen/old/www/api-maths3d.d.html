MTS Math3D Library :

This library is part of Standard (Std) library, so it is automatically included in all MTS files.

Vector :

Vector represents a point or a direction in 3D space.
new Vector() create a new vector.
new Vector(x,y,z) create a new vector.
_x x vector coordinate.
_y y vector coordinate.
_z z vector coordinate.
_length vector normalises.
+ add two vectors.
- substract two vectors.
* multiply the vector by a number or transform it by a matrix.
/ divide the vector by a number.
normalize() normalize the vector.

Matrix :

A matrix is a full representation of a 3D transformation.
new Matrix() create an identity matrix.
new Matrix(m) create a copy of the matrix m.
new Matrix(q) create a rotation matrix from the quaternion q.
new Matrix(t) create a matrix from the TRS t (see below for a description of TRS).
new Matrix(vt,qr,vs) create a matrix from the quaternion qr the translation vector vt and the scale vector vs (each value of this vector is the scale factor along an axis). Each argument accepts a null value.
* multiply (compose) two matrix.
inv() return matrix inverse.
pos() return the position vector included in the matrix.

Quaternion :

A quaternion represents a 3D rotation.
new Quaternion() create a new quaternion.
new Quaternion(q) create a copy of the quaternion.
new Quaternion(v,a) create a new quaternion from an angle a and a vector v representing the rotation axis.
new Quaternion(x,y,z) create a new quaternion with rotation values around x, y and z axis.
* combine two quaternions.
inv() inverse a quaternion.
to_matrix() return corresponding rotation matrix.

TRS :

A TRS (translation-rotation-scale) is a class allowing simplified matrix building, taking into account the three basics transformations separatly.
new TRS() create a new TRS.
new TRS(t) create a new TRS with its translation vector initialised to t.
new TRS(trs) create a copy of a TRS.
new TRS(vt,qr,vs) create a TRS from the quaternion qr the translation vector vt and the scale vector vs. Each argument accepts a null value.
_tx access to the translation x value.
_ty access to the translation y value.
_tz access to the translation z value.
_sx access to the scale x value.
_sy access to the scale y value.
_sz access to the scale z value.
_t read/write translation vector.
_r read/write rotation quaternion.
_s read/write scale vector.
+ , * combine two TRS.
to_matrix() compute the TRS corresponding matrix.

Additional functions :

This functions are part of the math library.
vlerp(v1,v2,f) do a linear interpolation between two vectors v1 and v2 according to f coefficient.
vdot(v1,v2) compute the dot-product of the two vectors.
vcross(v1,v2) compute the two vectors cross-product.
qslerp(q1,q2,f) do a spherical interpolation between two quaternions q1 and q2 according to f coefficient.
