Controls :

Key :

A Key Object represents a keyboard key or a mouse button.
new Key(name) create a new key corresponding to the keyname "name" (see below the keyname table).
new Key(code) create a new key corresponding to the keycode "code" (these codes are not ASCII codes).
name() return the key name, or null if no name correspond.
code() return the key code.
is_pressed() determine if the key have just been pressed.
is_down() determine if the key is actually pressed.
is_released() determine if the key have just been released.
time() return ellapsed time (in milliseconds) since the key was pressed (or 0 if the key is actually released).
_repeat read/modify necessary time before automatic key repetition. If negative, the key will nerver be repeated.

Mouse :

Mouse object is a global object part of the standard library (Std). If you define another global with the same name, you will always be abble to access the mouse with Std.mouse.
_x (read only) return mouse x coordinate. It corresponds to an integer number of pixels distance of the upper left corner of the clip.
_y (read only) return mouse y coordinate. It corresponds to an integer number of pixels distance of the upper left corner of the clip.
left left mouse button : it is a Key type object.
right right mouse button : it is a Key type object.
middle middle mouse button : it is a Key type object.
