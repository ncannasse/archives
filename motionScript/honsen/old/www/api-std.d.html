MTS Standard Library :

Standard functions :

print(...) display values in the log window.
display(...) display value in graphic window (erased at each frame).
random(x) return a random value between 0 and (x-1) included.
int(x) truncate x into an integer if float, or converts the string to an integer.
number(x) convert the string into a number (float or int).
to_string(x) convert x into a string.
compare(x,y) return 0 if x = y , -1 if x < y , 1 if x > y or null if x and y are not comparable.
_width return Honsen clip width in pixels.
_height return Honsen clip height in pixels..
time() return elapsed time in millisecond since clip start.
tick_time() return elapsed cpu time in millisecond since clip start.
set_main(f) change the MotionScript function called in the main loop.

List :

A linked list is very efficient for add push and remove operations. It is better to use it rather than an array in case of intensive use of theses functions. On the other hand, a list cannot be accessed using indexes ( with [n] ).
new List() create a new empty list.
add(x) add an element at the end of the list.
push(x) add an element at the beginning of the list.
remove(x) remove an element of the list. return true if x was found, null else.
pop() remove and return first list element.
_length (read only) return number of elements in the list.
iter(f,o) call f(x1,o) f(x2,o) ... f(xN,o) with x1..xN the elements of the list.
find(f) call f function with each element of the list in parameter until f returns not null.
to_array() return array containing same elements.

Array :

An array is very efficient for index access ( with [n] ) when its length is greater than n (in other case, the array automatically grows up but looses performances ).
new Array() create a new empty array.
new Array(n) create a new array of size n.
new Array(x,y) create a new bi dimentional array of size x, y.
new Array(x,y,z) create a new tri dimentional array of size x, y et z.
_length (read) return array size, (write) modify array size.
[n] access in read or write at array element at index n.
concat(...) add parameters to array. If one of the parameters is an array, its contents is added.
sub(start,len) return a sub-array from start to start + len.
sort(f_compare) sort arrays elements according to the comparaison function passed in parameter.
to_list() return a list containing same elements.

String :

A string is an object.
new String() create a new empty string.
new String(s) create a new string, copy of s.
[n] (read only) return character at position n in the string.
_length (read only) return the string length.
sub(start,len) return a substring in the form of a new string. if len = null returns s.sub(start,s._length-start).

Event :

An event is a function called at regular interval.
new Event(f,t) create a new event which will call f function each t milliseconds.
start() start event execution. By default, an event is stopped.
stop() stop the event execution. It can be restarted later.
running() return true if event is started, false if not.
_time (read and write) : read/modify event execution time interval, in milliseconds. This time have to be positive.
_callback (read and write) : read/modify event function called at each interval. This function cannot be null.
