Honsen Documentation

Honsen documentation is split in different parts. This way, you might read the documentation appropriate to your speed of assimilation :

  • Honsen Reference [html]
    This documentation explains all about Honsen SDK Tools and how to create honsen contents.
  • MotionScript Language Reference [html]
    This documentation explains all about the MotionScript programming language used to write Honsen programs.
  • Honsen API Reference [html]
    This documentation explains all the APIs that are available from MotionScript.
  • Honsen Tutorials [html]
    Want to learn Honsen quickly ? From "Hello World" to complex game, a full range of tutorials to get a taste of Honsen.
