Honsen main features

Here are some of the main features of Honsen :

  • ActiveX Browser Plugin :
    • for Internet Explorer 4 or later
    • ... soon for Mozilla/Firefox
  • 3D Engine :
    • DirectX 8 based
    • Camera and Light control
    • 3D accelerated Sprites and Fonts
    • Alpha and color effects
    • Customizable pipeline
    • Shaders support
    • Mesh Skinning (bones based animation)
  • MotionScript programming language :
    • Common usage syntax (C style)
    • Easy to use, easy to learn APIs :
      • 3D Maths API (Vector, Matrix, Quaternion)
      • Graphics API (Camera, Meshes, Sprites, Text)
      • Input API (Mouse and Keyboard)
    • Dynamic typed robust language
    • Fast and secure virtual machine
    • Object Oriented features
    • Separate compilation
    • Advanced OO features : operator overloading, properties...
  • Free Authoring SDK including :
    • open source 3D import file formats
    • open source Blender Python export plugin
    • MotionScript compiler
    • Standalone executable player
    • HS file builder
    • 3D models and animation viewer
    • ... all the tools you need to create Honsen content
