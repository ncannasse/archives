3D library :

Mesh:

A mesh is a 3D object.
new Mesh(r) add a new 3D object to the scene, r indicate an object resource.
new Mesh(r,p) add a new 3D object to the scene, r indicate an object resource and p the parent mesh.
_visible read/modify the object visibility.
_pos read/modify the object position : this position is a matrix, nevertheless a TRS could be written in _pos (it will be automatically convert to a matrix).
_color lread/modify the object color.
_alpha read/modify the object transparency.
_texture read/modify the object texture (a null value allow to destroy the texture).
_parent (read only) : return the parent object of the Mesh or null if there is no.
remove() delete the mesh from the scene.
move(x,y,z) move the mesh (modify _pos).
scale(f) change mesh size (modify _pos).
rot(x,y,z) rotate the mesh (modify _pos).
_shader read/write the vertex shader used to display this mesh. Shader is a string containing assembly code (Honsen supports shaders up to v1.1).
has_tag(t) return true if the mesh contains the t tag and false else (see MTW file format for more details on tags).

Camera:

camera globale give you access to the camera (or Std.camera).
_pos position vector of the camera.
_target camera target vector.
_world general world matrix.
_proj the projection matrix.
_roll read/modify the camera roll value (radians)
_zoom read/modify actual zoom value (back to one at each _pos modification).

Light:

light globale give you access to the light illuminating the scene.
_direction read/write light direction vector.
_color read/write light color on 6 bits(RGB).
_ambient read/write world ambient color on 6 bits(RGB).
