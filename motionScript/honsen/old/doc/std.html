MTS Standard Library :

Fonctions standard :

print(...) affiche des valeurs dans la fenêtre de log.
display(...) affiche des valeurs dans la fenêtre graphique (effacées chaque frame).
random(x) renvoie un entier entre 0 et (x-1) inclus.
int(x) tronque x en entier si flottant, ou convertis la chaine de charactères en entier.
number(x) convertis la chaine de charactères en nombre (float ou int).
to_string(x) convertis x en chaine de charactères.
compare(x,y) retourne 0 si x = y , -1 si x < y , 1 si x > y ou null si x et y ne sont pas comparables.
_width retourne la longueur en pixels du clip.
_height retourne la hauteur en pixels du clip honsen.
time() retourne le temps écoulé depuis le démarrage du clip.
set_main(f) change la fonction MotionScript qui sera appellée dans la boucle principale.

List :

Une liste chainée est trés efficace pour les opérations add, push et remove. Il est donc préférable d'utiliser une liste plutot qu'un Array dans les cas ou ces opérations sont beaucoup utilisées. Une liste par contre ne peut pas être accédée par index ( avec [n] ).
new List() crée une nouvelle liste vide.
add(x) ajoute un élément à la fin de la liste.
push(x) ajoute un élément au début de la liste.
remove(x) supprime un élement de la liste. renvoie true si x était dans la liste, null sinon.
pop() supprime et retourne le premier element de la liste.
_length (lecture) renvoie le nombre d'élements de la liste.
iter(f,o) appelle f(x1,o) f(x2,o) ... f(xN,o) avec x1..xN étant les éléments de la liste.
find(f) appelle la fonction f avec chacun des éléments de la liste jusqu'à ce que f retourne non null.
to_array() retourne le tableau contenant les mêmes éléments.

Array :

Un tableau est trés efficace pour les accés par index ( avec [n] ), surtout quand la taille du tableau est superieure à n (dans le cas contraire, le tableau se redimentionne automatiquement mais cela à un coup en performances).
new Array() crée un nouveau tableau vide.
new Array(n) crée un nouveau tableau de taille n.
new Array(x,y) crée un nouveau tableau à deux dimensions de tailles x et y.
new Array(x,y,z) crée un nouveau tableau à trois dimensions de tailles x , y et z.
_length (lecture) retourne la taille du tableau, (ecriture) modifie la taille du tableau.
[n] accéde en lecture ou écriture au n-ième élément du tableau.
concat(...) ajoute les paramètres au tableau. Si un de ces paramètres est un tableau, son contenu est ajouté.
sub(start,len) retourne une sous partie d'un tableau sous la forme d'un nouveau tableau.
sort(f_compare) tri les élements du tableau d'aprés une fonction de comparaison.
to_list() renvoie la liste contenant les mêmes éléments.

String :

Une chaine de caractères est aussi un objet.
new String() crée une nouvelle string vide.
new String(s) crée une nouvelle string qui est une copie de s.
[n] (lecture seulement) retourne le caractère de la string à la position n.
_length (lecture seulement) retourne la taille de la chaine de caractères.
sub(start,len) retourne une sous partie d'une string sous la forme d'une nouvelle string. si len = null alors retourne s.sub(start,s._length-start).

Event :

Un évènement est une fonction appellée qui se déclenche à un intervalle régulier.
new Event(f,t) crée un nouvel évènement qui appellera la fonction f toutes les t millisecondes.
start() démarre l'exécution de l'évènement. Lors de sa création, un évènement est arrété.
stop() stoppe l'exécution d'un évènement. Ce dernier pourra être redémarré ensuite.
running() retourne true si l'évènement est actuellement démarré, false sinon.
_time (lecture et écriture) : lis/modifie l'intervale d'exécution de l'évènement, en millisecondes. Ce temps doit être strictement positif.
_callback (lecture et écriture) : lis/modifie la fonction appellée a chaque intervalle. Cette fonction ne peut être null.
