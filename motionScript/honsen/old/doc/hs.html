Honsen Tutorial :

Les Fichiers :

Quelques indications sur les différents formats de fichiers utilisés par Honsen:
.hsx (equiv .fla) Le Bibliothèque d'un clip honsen : elle décris quels fichiers vont être ajoutés au clip, l'ordre de leur chargement, les fichiers sources qui vont être executés, etc. (voir ci-dessous la description du format HSX)
.hs (equiv .swf) Le fichier HS est un Clip Honsen. Il contiens toutes les informations nécessaires à son execution et peut etre lancé avec le Player Honsen.
.mts (equiv .as) Un fichier MTS contiens du code source Motion-Script.
.mto Une fois compilé, le fichier MTS produit un fichier MTO du même nom.
.mtw Un fichier MTW est exporté depuis 3DSMax : il s'agit d'un objet 3D pouvant contenir des animations. Il peut être ajouté à la Bibliothèque.
.jpg .png Les fichiers JPEG et PNG sont des images qui peuvent être ajoutées à la Bibliothèque et serviront de texture pour les modèles 3D ou d'image pour afficher les sprites.

La Bibliothèque :

La Bibliothèque rassemble tous les éléments nécessaires à la création d'un clip HS. Elle est décrite dans un fichier au format HSX : 
	<honsen width="400" height="300" bgcolor="0xFFFFFF" fps="40">
	    <mts file="test.mts"/>
	    <pic file="particule.png" name="Particule"/>
	    <mtw file="cube.mtw">
	    	<id object="cube3D" name="Cube"/>
	    </mtw>
	    <font face="Times New Roman" size="16" bold="true" italic="false" name="Times16"/>
	</honsen>
width la longueur en pixels du clip Honsen.
height la hauteur en pixels du clip Honsen.
bgcolor la couleur (RGB) de fond du clip Honsen.
fps la vitesse d'execution de la boucle main (code Motion-Script).
mts un fichier source Motion-Script est ajouté à la Bibliothèque.
pic une image est ajoutée à la Bibliothèque.
mtw un objet 3D est ajouté à la Bibliothèque.
font un nom de font est ajouté à la Bibliothèque. Pour ajouter une font qui n'est pas une font système, il faut définir l'attribut "file": le fichier correspondant sera automatiquement ajouté la Bibliothèque.
name le champ "name" pour les fichiers "pic", et "font" désigne un nom de resource qui permettra par la suite d'accéder au données depuis Motion-Script (voir l'aide Motion-Script sur les resources).
pour les fichiers "mtw", cette resource doit être attachée à un objet 3D contenu dans le fichier.

Créer un Clip :

Une fois la Bibliothèque au format HSX créee, l'outil hsmake.exe permet de créer le Clip Honsen correspondant. Pour cela, il suffit d'éxecuter hsmake file.hsx. Si tout se passe bien, un fichier file.hs est crée qui pourra être lu avec le Player Honsen. Sinon, un message erreur est affiché. L'outil hsmake.exe effectue les tâches suivantes :
  1. vérifie la validité du fichier HSX
  2. vérifie l'existence et le format de tous les fichiers de la bibliothèque
  3. compile tous les fichiers Motion-Script de la bibliothèque
  4. rassemble les fichiers MTO compilés en un fichier MTX
  5. crée un clip HS qui rassemble tous les fichiers de la bibliothèque

Lire un Clip :

Une fois le clip HS crée, il suffit de lancer le Player Honsen (hsrun.exe) pour l'exécuter.
