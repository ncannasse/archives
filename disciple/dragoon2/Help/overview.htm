Overview

When you Start Dragoon, you can see the following things :

  • An icon-bar to the left.
    This bar are shortcuts of Dragoon's / Plugin's fonctionnalities. You can click on them to create folders. For easy use, you can access all the fonctionnalities by four ways :

    • Icon click

    • Right Mouse Button Popup menu

    • Window Menu click

    • KeyBoard shortCut.

  • A form called " Console "
    Here are logged all messages from Dragoon's or Plugin's. The Console move under the Project Workspace when you load one. If a message you want read excess the Console Width, just DoubleClick on it and he will appear in a MessageBox.

  • The Fast-Load Menu
    The last three shortcuts for your recents projects.
