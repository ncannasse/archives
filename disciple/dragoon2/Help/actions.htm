    • ACTIONS:
      • ReceiveActions

        typedef struct
        {

          DWORD Code;
          DWORD ID;
          char *Name;
          char *Author;
          char *Help;
          HANDLE hInst;

          PLUGPROC ReceiveActions;
          DWORD Reserved[4];

        } TPlugInfo, *LPPlugInfo

        The ReceiveActions Field of the TPlugInfo struct can be set to one Plugin-local fonction of type PLUGPROC ( take 2 void* and return a bool ). So, when loading the project, Dragoon will call this fonction, passing it a pointer to a TActions struct

        typedef struct TActions {


          CREATEOBJPROC CreateObj;
          PLUGPROC DelObj;

          PLUGPROC SetObjVisible;
          PLUGPROC SetObjEnabled;

          PLUGPROC SetCaption;

          PLUGPROC ListAddItem;
          PLUGPROC ListDelItem;
          PLUGPROC_EN ListGetItemByIndex;
          PLUGPROC_EN ListGetItemByName;
          PLUGPROC ListClear;

          PLUGPROC GridClear;
          PLUGPROC_EN GridSetCount;
          PLUGPROC_EN GridSetLength;
          PLUGPROC_EN GridSetCell;
          PLUGPROC_EN GridGetCell;

          PLUGPROC EditSetText;

          PLUGPROC_EN InputTxt;
          PLUGPROC Log;
          PLUGPROC CnvIDToFileName;
          PLUGPROC RunPAKBrowser;
          PLUGPROC ShowPlugForm;
          PLUGPROC_EN RunPlugin;

          PLUGPROC CreateItem;

        } TActions, *LPActions;


        All the fields of this fonction will be specified later, and parameters to pass will be specified.
        The ReceiveActions procedure is used to get a Handle to a valid TActions struct.
        This Handle will be use for further calls to Dragoon's API.

        What to Do ?
        Set a global LPActions var to the first parameter passed

        What does it return ?
        no errors are handling, so you can return any value
