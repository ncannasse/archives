  • Plugin Initialisation

    • The PlugGetInfo Function

      bool PluginGetInfo( LPPlugInfo inf, void * );

      This Fonction is used by Dragoon to initialise the Plugin. Dragoon create a new TPlugInfo struct and call PluginGetInfo by passing a pointer of it. The second parameter is not used.

      What to Do ?
      PluginGetInfo MUST fill the fields of the TPlugInfo struct, or return false

      What does it Return ?
      PluginGetIngo must return true if the Plugin have been correctly installed, or false if it failed.
      In this case, the plugin will be free from memory and no other call will be done.
