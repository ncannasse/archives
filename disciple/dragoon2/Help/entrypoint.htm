    • The plugin EntryPoint

      This is the standard Microsoft Visual Studio DLL Entry-Point. If you're not using Visual Studio, use the DLLEntryPoint specified in your developpement system's help:

        BOOL APIENTRY DllMain(HANDLE hModule, DWORD ul_reason_for_call,LPVOID lpReserved)
        {
          switch( ul_reason_for_call )
          {
          case DLL_PROCESS_ATTACH:
            // perform any memory allocations / inititialisations
            // you can store the hModule param, it will be used
            // for HINSTANCE in LoadBitmap function
             break;
          case DLL_PROCESS_DETACH:
            // free memory
             break;
          default:
            break;
          }
          return TRUE;
        }


      • See Windows 32 SDK Help for more info on the meaning of this function.
