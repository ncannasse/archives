Files / Folders Organisation

Dragoon has a WindowsExplorator-Like Files / Folders virtual organisation.
Your project WorkSpace can be see like that :

  • The main folder (here : <test.prj> ) is the name of your project. You can create subfolders by clicking on the icon "NewFolder" ( see below ) of by running the popup menu with a simple right-click on the Workspace
  • You can create ITEMS ( means perhaps a file - like a PAK, or else a logical unit not physically present ) by clicking on the icons of the Ico-Bar to the left or by running the popup Menu.
  • You can organize easily your items and your folders by a simple Drag'n'Drop on your Workspace. As Folder implementation is completely hidden toPlugins, they can't interfer with them.
  • Cut / Paste with Ctrl-X / Ctrl-V is also accessible
  • You can also Rename / Delete the items with the popup RButton menu. or with F2 / Del keyboards keys
