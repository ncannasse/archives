      • OnLoadProjectFinished

        typedef struct
        {

          DWORD Code;
          DWORD ID;
          char *Name;
          char *Author;
          char *Help;
          HANDLE hInst;

          PLUGPROC ReceiveActions
          PLUGPROC OnPAKChange;
          PLUGPROC OnLoadProjectFinished;

          DWORD Reserved[2];

        } TPlugInfo, *LPPlugInfo;

        The OnLoadProjectFinished field of the TPlugInfo struct can be set to handle the event wich occurs when the user load the project for the first time.
        The first parameter passed to this fonction is a NULL-terminated String which contains the Project Full-Path name that have been loaded. The second parameter is always NULL.

        What to Do ?
        Load your own data file

        What does it return ?
        what you want.
