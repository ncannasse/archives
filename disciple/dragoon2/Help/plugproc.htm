    • PLUGPROC Functions

      typedef       bool (*PLUGPROC)( void * = NULL, void * = NULL);
      typedef bool (*PLUGPROC_EN)( void * = NULL, void * = NULL, void * = NULL );

      These are two main standard fonctions for communication between Dragoon and his Plugins.
      a PLUGPROC fonction takes two parameters, wich default values are NULL if non specified at call's time,
      a PLUGPROC_EN does the same, but with three parameters.
