    • TPlugInfo struct

typedef struct
{

    DWORD Code;
    DWORD ID;
    char *Name;
    char *Author;
    char *Help;
    HANDLE hInst;

    DWORD Reserved[5];
} TPlugInfo, *LPPlugInfo;

Fields description:

  • Code: The plugin must set this field to CODE ( 0xDAFF00CE ) if he want be recognized as a Dragoon's plugin.
  • ID: This is the Identification of a plugin. ( ex: Animator has code 1 ). It's reserved for future use, like Inter-Plugin Advanced Communications.
  • Name: Pointer to a NULL terminated String that contains the full Name of the Plugin ( ex: "Animator v1.34")
  • Author: Pointer to a NULL terminated String that contains the name of the Author of the PlugIn
  • Help: Pointer to a NULL terminated String that contains a small text that we'll be display has an help / info for this plugin
  • hInst: the Plugin HANDLE, used like HINSTANCE for LoadBitmap fonctions.
  • Reserved: These fields are not explicited here. See Later in the Reference to use thems.
