Dragoon's Plugin Dev OnLine Help

 

Dragoon is a base platform of developpement for 2-Dimentions VideoGames, created by Nicolas CANNASSE ( Warplayer ). Developped only for Windows ( 95 - 98 , no NT ), it's an easy way to merge and manage your own tools for your games.

Good tools can help you a lot. They have never make good games, but they can let you the time for working on the real sense of a Video Game : the GamePlay ( or, programmers will say, a full debugging of source code ).

Menu:

- Overview
- Project Management
- Files / Folders organisation
- Files Types own definition
- PAK Files full gestion
- Plugin Developpement

These standard Plug-Ins are includes in the package :

- Symbols Editor
- Animator
